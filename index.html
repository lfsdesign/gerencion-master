❓ Como usar o Painel Master
🔐
O que é o Painel Master?
É a sua central de controle. Só você tem acesso. Aqui você vê todos os clientes, ativa ou desativa o acesso, define o plano e a validade de cada um.
➕
Como ativar um novo cliente — passo a passo
1. Cliente paga o plano combinado
2. Ele acessa o app e cria a conta (gerencion-estoque.vercel.app)
3. Ele aparece aqui na lista automaticamente
4. Você clica em ✏️ Editar no card dele
5. Define o Status como ✅ Ativo
6. Escolhe o plano correto
7. Coloca a data de validade (ex: 30 dias a partir de hoje)
8. Clica em Salvar ✅
📋
Botão "Copiar Código" — para que serve?
Cada cliente tem um código de 6 letras único (ex: AB1C2D). Você copia esse código e envia por WhatsApp para o cliente. Ele abre o app, clica em "Entrar com Código de Acesso", digita o código e já entra no sistema dele.
✏️
Botão "Editar" — o que posso mudar?
• Status: Ativo ou Inativo (corta acesso na hora)
• Plano: Básico, Profissional ou Empresarial
• Validade: até quando o cliente tem acesso
• Máx. Funcionários: quantos funcionários ele pode cadastrar
• Observação: anote qualquer coisa (ex: "pagou Pix dia 05", "Cliente Fundador")
⛔
Botão "Desativar" — quando usar?
Quando o cliente não pagou ou pediu cancelamento. O acesso é bloqueado na hora. Para reativar é só clicar em ✅ Ativar no mesmo botão.
🔄
Botão "+30 dias" — como funciona?
Adiciona 30 dias à validade atual do cliente. Se ele já está vencido, conta 30 dias a partir de hoje. Ativa o cliente automaticamente. Use quando o cliente renova o plano.
⭐
Badge "Cliente Fundador" — como aparece?
Clique em ✏️ Editar no card do cliente e escreva "Cliente Fundador" no campo Observação. O badge dourado aparece automaticamente no card dele.
🗑️
Excluir cliente — quando usar?
Use para remover clientes de teste ou cancelados definitivamente. Atenção: esta ação remove todos os dados do cliente e não pode ser desfeita!
🔍
Filtros e busca
Use a barra de busca para encontrar um cliente pelo nome. Use os botões de filtro para ver apenas Ativos, Inativos, Vencendo ou por plano.
